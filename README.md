# Modtrix

一个通用的Mod管理器，使用electron构建，参考了motrix的UI设计。

[![preview02.png](https://i.postimg.cc/hPW3vbC8/preview02.png)](https://postimg.cc/Th0cHmvh)

## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
```

### Build

```bash
# For windows
$ npm run build:win
```
