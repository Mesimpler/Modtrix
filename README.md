# Modtrix

一个通用的Mod管理器，使用electron构建，参考了motrix的UI设计。

> [![preview01.png](https://i.postimg.cc/tg7nXBB5/preview01.png)](https://postimg.cc/GBwmKxw8)

## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
```

### Build

```bash
# For windows
$ npm run build:win
```
